First level: Дайте ответы на след/вопросы:

    1. Что такое Linux и для чего он применяется.
    - Програмное обеспечивание с открытым исходным кодом.
    - применяется для серверов, мобильных устройств, IoT, Рабочих станций/ стаяионарных компьютеров

    2. Чем отличается CLI от GUI.
    - CLI это интерфейс командной строки -> более практичный режим работы
      в отличии от GUI который является графическим интерфейсом пользователя

    3. Что такое SSH.
    - это ключи для безпароленного входа/ подключения к серверу и GitHub

    4. Как сгенерировать пару RSA-ключей и зачем это делать.
    - использовать команду ssh-keygen и несколько раз enter -> ключ сгенерировался
    - для быстрой загрузки файлов на GitHub, соединения с сервером

    5. каково назначение команды df ?
    - disk free - статистика по дискам

    6. как посмотреть публичный ключ на своем компьютере? 
    - cat ~/.ssh/id_rsa.pub

    7. Какие задачи решаются с помощью команды 'gzip'
    - сжатие/ компрессия файла
